**FEM_PlateBending** is a Python implementation of the Finite Element implementation applied to a Kirchoff plate subjected to forces acting normal to its mid-surface plane. The theoretical background is based on the book "The Implementation of the Finite Element Method" by Viktor Hristovski. FEM_PlateBending is organized as follows:

The Jupyter Notebook "FEM_PlateBending.ipynb" is a step-by-step guide intended to explain the theoretical background and its Python implementation. Its building blocks are explained in each step, and are followed by a code section in which the reader can directly relate how is the code interpreting the theoretical background. The reader can easily try its own examples by modifying the desired geometrical and mechanical properties as well as the problem's boundary conditions.
